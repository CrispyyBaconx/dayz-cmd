# Gap Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the highest-value gaps from the review by making direct connect truly direct, exposing missing server-filter and BattleMetrics actions, and turning launch options into a real editor with test coverage.

**Architecture:** Keep the existing single-binary Rust TUI shape and patch the weakest seams instead of refactoring the app. Add narrowly scoped helpers around launch targets, filter state, and launch-option editing so behavior becomes testable without widening the surface area.

**Tech Stack:** Rust, ratatui, clap, serde, std fs/process, cargo test
